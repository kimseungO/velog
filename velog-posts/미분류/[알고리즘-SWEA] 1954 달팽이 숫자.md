SWEA 의 D2문제. 달팽이처럼 증가하는 숫자로 원을 그린다.
ex) 
1 2 3
8 9 4
7 6 5

D2문제 치고는 난이도가 상당했다.

재귀문으로 푸는데, 꺾이는 구간을 일일이 조건문을 넣어줘야 했다.

시간이 굉장히 오래걸렸다.

### 정답 코드
```python
import sys
sys.stdin = open("input.txt", "r")

def dfs(x, y, n, dir): # 깊이 탐색 함수
    dx = [0, 1, 0, -1] # 방향 백터
    dy = [-1, 0, 1, 0]

    while 0 <= x < N and 0 <= y < N and n<N**2: # 현재 좌표가 맵 안에 있고 현재 넣어야할 숫자가 최대수를 벗어나지 않았을 때
        x = x + dx[dir%4] # 좌표에 현재 방향 백터만큼 더해주고
        y = y + dy[dir%4]
        n+=1 # 넣어야할 숫자를 더해줌
        snail[y][x]=n # 넣어야할 숫자를 현재 좌표에 넣어쥼
        if dir%4 == 1: # 오른쪽       # 오른쪽으로 가는 경우일 때
            if x == N-1 or snail[y][x+1] != 0: # 만약 좌표의 끝이라면 그리고 다음 좌표가 0이 아니라면(이미 숫자가 채워져 있다면)
                break # 현재 방향으로 진행은 종료
        elif dir%4 == 2: # 아래쪽
            if y == N-1 or snail[y+1][x] != 0:
                break
        elif dir%4 == 3: # 왼쪽
            if x == 0 or snail[y][x-1] != 0:
                break
        elif dir%4 == 0: # 위쪽
            if y == 0 or snail[y-1][x] != 0:
                break

    if n >= N**2: # 만약 넣어야할 숫자가 최대숫자가 되면
        return # 끝
    dfs(x, y, n, dir+1) # 아니면 현재 좌표와 넣어야할 숫자를 가지고 다음 방향으로 재귀

T = int(input())
for test_case in range(1, T + 1):
    N = int(input())
    snail = [[0 for x in range(N)] for x in range(N)] # 달팽이 초기화
    snail[0][0] = 1 # 1을 일단 넣어주기
    dfs(0, 0, 1, 1) # 재귀 함수에 현재 좌표(0,0)과 넣어야할 숫자(1) 방향(1)
    print(f"#{test_case}")
    for row in snail:
        print(*row)

```
