인덱스를 이용해서 푸는 문제
맥스 인덱스를 지정해 두고 for문을 돌면서 맥스 인덱스보다 크다면 맥스 인덱스를 업데이트 해준다.


https://www.youtube.com/watch?v=VRvZuMfP20k

### 코드 풀이
```python
T = int(input()) # 테스트 케이스
for test_case in range(1, T + 1): 
    n = int(input()) # 며칠?
    n_list = list(map(int, input().split())) # 날짜별 시세 입력
    s = 0 # 시작 인덱스 초기화
    ans = 0 # 정답 초기화
    while s<n:
        n_max = s # 맥스인덱스를 구하기위한 초기화
        for i in range(s+1, n): # 1~n의 맥스인덱스 구하기
            if n_list[n_max] < n_list[i]: # 만약 맥스인덱스보다 크다면
                n_max = i # 너가 맥스인덱스
        for i in range(s, n_max): # 시작 인덱스부터 맥스 인덱스까지
            ans += (n_list[n_max] - n_list[i]) # 정답 계산
            
        s = n_max+1 # 맥스 인덱스 다음 인덱스부터 while문 반복
    print(f"#{test_case} {ans}")
```
