문제만 보면 간단해 보였으나 생각을 좀 해야 되는 문제였다.

### 풀이
이 문제는 '-'뒤에 '+'구간은 무조건 괄호를 쳐서 먼저 계산을 하면 최소값이 나온다.
즉 식이 주어지면, split('-')으로 마이너스를 기준으로 나누어 플러스를 먼저 계산해준다.
split('+')으로 플러스를 기준으로 나누어 일단 string 형식을 int 형식으로 바꿔주면 '000007'과 같은 숫자는 7로 간단히 바뀌어 계산할 수 있다.

### 코드
```python
n = input()
m = n.split('-') # 마이너스를 기준으로 나눠줌

for i in range(len(m)): 
    m[i] = m[i].split('+') # 플러스를 기준으로 나눠서
    for j in range(len(m[i])): # 각 숫자를 int 형식으로 변환
        m[i][j] = int(m[i][j])
    m[i] = sum(m[i]) # 그 후 sum함수로 더해줌
ans = m[0] # 첫 숫자는 그냥 넣어주고
for i in range(1, len(m)): # 그 뒷 숫자부터 마이너스 해줌
    ans -= m[i]
print(ans)
```
