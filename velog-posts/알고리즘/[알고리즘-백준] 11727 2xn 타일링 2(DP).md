### 풀이
이 문제는 dp 문제로 점화식을 구해야 한다.

a1 = 1
a2 = 3
a3 = 5
a4 = 11
...
an = a(n-2)x2 + a(n-1)

원래는 문제를 보고 점화식을 도출해야 하지만
나열된 수를 보고 점화식을 발견했다.

그 후 풀이를 찾아서 보니 왜 이렇게 되는지 이해가 되었다.
참고: https://cijbest.tistory.com/21

### 정답 코드
```python
n = int(input())
dp = [0]*1001
dp[1]=1
dp[2]=3

if n>2:
    for i in range(3, n+1):
        dp[i] = dp[i-2]*2 + dp[i-1]

print(dp[n]%10007)
```
