### 풀이
전형적인 수열 문제.
수열을 쭉 나열해 보면 
{1, 1, 1, 2, 2, 3, 4, 5, 7, 9, 12, 16, 21, 28, 37, 49, ...}
이 때 이 수열은 설명에 있는 그림을 잘 보면 주기가 정해져 있다. 5개의 수 마다 한 바퀴.
그걸 이용해서 더해지는 값을 보면
{-, -, -, -, -, 1, 1, 1, 2, 2, 3, 4, 5, 7, 9, 12, ...}
즉, 현재의 주기는 이전 주기의 수를 그대로 차례로 더한 값

### 정답
```python
t = int(input())
for i in range(t):
    n = int(input())
    dp = [1,1,1,2,2]
    if n<5:
        print(dp[n-1])
    else:
        for j in range(5, n):
            dp.append(dp[j-1] + dp[j-5])
        print(dp[n-1])
 ```
