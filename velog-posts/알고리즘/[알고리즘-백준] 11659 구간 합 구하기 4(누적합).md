문제만 보면 쉽다고 생각했지만 예상대로 호락호락하진 않았다.
단순 구현이 아닌 누적합 문제였고, 단순히 구현할 시에 시간 초과로 실패했다.

### 풀이
누적합은 수열이 있으면 그 수열의 합을 나열한 또 다른 수열을 이용한다.
수열 예시)
{5, 4, 3, 2, 1}
누적합 예시)
{5, 9, 12, 14, 15}

이를 이용해 풀었을 때 어렵지 않게 풀었다.

### 정답
```python
import sys
input = sys.stdin.readline

n, m = map(int, input().split())
num = list(map(int, input().split()))

prefix = [0]
for i in range(n):
    prefix.append(prefix[i]+num[i])

for _ in range(m):
    i, j = map(int, input().split())
    res = prefix[j]-prefix[i-1]
    print(res)
```
