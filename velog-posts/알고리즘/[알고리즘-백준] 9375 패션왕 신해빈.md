### 문제
해시와 수학을 이용해 푸는 문제
딕셔너리의 밸류를 기준으로 그룹화하여 각 밸류가 등장하는 횟수를 계산하려면 collections 모듈의 Counter를 사용하는 것이 간단하다.

### 풀이
```python
from collections import Counter

test_case = int(input())

for i in range(test_case):
    n = int(input())
    wardrobe = {} # 옷장
    for i in range(n):
        cloths, types = input().split()
        wardrobe[cloths] = types # 옷장에 옷들 넣기
    value_cnt = Counter(wardrobe.values()) # 각 옷의 타입별로 그 수를 계산

    cnt = 1
    for i in list(value_cnt.values()): # 계산한 값에 접근(딕셔너리.values())
        cnt *= (i+1)
    print(cnt-1)
```
