이진탐색 문제.
이진탐색은 탐색할 수가 많을 때 중간부터 탐색할 수까지 탐색거리를 좁혀나가는 방식이다.

문제를 보고 이진탐색을 잠깐 떠올렸으나 헤매다가 시간이 오래걸린 문제..

### 풀이
```python
n, m = map(int, input().split())
tree = list(map(int, input().split()))

start =0 
end = max(tree)
res = 0 # 최종 절단기 높이
while start<=end:
    total = 0 # 가져갈 나무의 길이
    mid = (start+end)//2 # 현재 시도하는 절단기 높이
    for i in tree: # 현재 절단기로 잘랐을때 가져가는 나무의 길이 계산
        if i > mid:
            total += i -mid
    if total > m: # 만약 가져갈 나무의 길이가 m보다 크다면
        res = mid # 절단기 높이 mid로 설정
        start = mid+1 # mid 기준 오른쪽 부분 탐색
    elif total < m: # 만약 가져갈 나무의 길이가 m보다 작다면
        end = mid-1 # mid 기준 왼쪽 부분 탐색
    elif total == m: # 만약 가져갈 나무의 길이가 m과 일치한다면
        res = mid # 절단기 높이 mid로 설정
        break # 반복문 종료
print(res)
```
